# Python - Classes and Objects

## Introduction
This project introduces the basics of **Object-Oriented Programming (OOP)** in Python.  
You will learn how to create classes, instantiate objects, use attributes, and define methods.

---

## Learning Objectives
- Understand what a **class** is  
- Understand what an **object** is  
- Know the difference between **class attributes** and **instance attributes**  
- Understand the role of the `__init__` method  
- Learn how to create attributes and methods  
- Understand the purpose of `self`  
- Learn how Python represents objects  
- Understand how OOP organizes and structures code
