## Description
This project covers **Inheritance in Python**, one of the main concepts of Object-Oriented Programming (OOP).
Inheritance allows a class to inherit attributes and methods from another class, helping reduce code duplication
and improve code organization.

---

## Learning Objectives
By the end of this project, you should be able to:

- Understand what inheritance is
- Explain parent and child classes
- Understand how Python searches for attributes and methods
- Use built-in functions related to inheritance
- Use `isinstance()`, `issubclass()`, and `dir()`
- Override inherited methods
- Understand the role of the `object` class
