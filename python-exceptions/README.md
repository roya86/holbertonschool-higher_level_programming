# Python - Exceptions

## Description
This project introduces how to handle errors and exceptions in Python.  
You will practice using try/except, raising exceptions, and writing safe functions that avoid program crashes.

## Learning Objectives
- What exceptions are and why they occur
- How to handle exceptions using try and except
- How to use else and finally
- How to raise exceptions
- How to write functions that safely handle errors
- How to print errors to stderr
- How to create safe operations without using built-in helpers
