# Python - Import & Modules

This project covers how to import functions, create modules, use `dir()`, and handle command line arguments in Python.

## Learning Objectives
- Import functions from another file
- Create and use modules
- Use `dir()` to list names
- Prevent code from running when imported
- Use command line arguments with `sys.argv`

## Requirements
- Ubuntu 22.04, Python 3.10
- First line: #!/usr/bin/python3
- Files must be executable
- Follow pycodestyle 2.7.*
- All files end with a new line
- README.md required
- Length checked with `wc`
